## Hacker News/Technical

**Show HN: dotclaude - Claude Code agents with security-focused prompt sanitizer**

Repository: https://github.com/ngmisl/dotclaude

A curated collection of Claude Code development guidelines (CLAUDE.md), specialized agents, and a Go-based prompt sanitizer.

The sanitizer (214 LOC, stdlib only) scans markdown/yaml/xml/txt files for:
- Unicode zero-width characters (U+200B, U+200C, etc.)
- Bidirectional text overrides (U+202E, etc.)
- Homoglyphs (Cyrillic '–µ' vs Latin 'e')
- Base64-encoded payloads with keyword detection
- Invalid UTF-8 and control characters

Built this after realizing shared prompts/agents could hide malicious instructions invisible to humans but parsed by LLMs. The agent collection includes code quality checkers, debugging agents, and compliance validators.

Opinionated but practical. Actively maintained.
